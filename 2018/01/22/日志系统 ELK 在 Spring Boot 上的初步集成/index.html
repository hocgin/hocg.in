ELK

Elasticsearch + Logstash + Kibana，一个比较流行的日志系统。当然，你也可以尝试使用阿里云。如果钱多的话:)

Logstash

日志搬运, 可以指定从 Redis/Log4j..处获取日志信息，然后进行过滤，再提交给 Elasticsearch 进行分析。

Elasticsearch

日志存储 / 查询 / 分析
